Bom mostro os resultados das minhas revisões de ferramentas para a Analise de Dados.
Primeiro, desenvolvi um sistema Python integrado ao SQL, como reforço da revisão. O sistema adiciona informações no SQL Server que por sua vez retorna ao Python que cria dashboard através de bibliotecas. 
Usei estes frameworks:
Dash, constroi aplicativos web analisticos e interativos em python;
Pandas é uma biblioteca Python para manipulação e analise de dados;
PyODBC, responsavel por integrar o Python ao SQL Server.
